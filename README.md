# 신성대학교 평면도 검색 프로그램

이 프로그램은 신성대학교의 건물명과 호실 정보를 검색하여 해당 도면을 보여주는 웹 애플리케이션입니다.

## 사용 방법

1.  `shinsung-map-search` 폴더를 엽니다.
2.  `index.html` 파일을 브라우저(Chrome, Edge 등)로 엽니다.
3.  검색창에 `본관 201`, `공학관 1F` 등 원하는 정보를 입력합니다.

## 이미지 설정 (중요)

보내주신 이미지들을 아래의 규칙에 따라 `assets/plans/` 폴더 안에 저장해 주세요:

-   **본관 A-1층** 이미지 -> `assets/plans/page_1.png`
-   **본관 A-2층** 이미지 -> `assets/plans/page_2.png`
-   **본관 A-3층** 이미지 -> `assets/plans/page_3.png`
-   **공학관 B-1층** (나중에 구하게 되면) -> `assets/plans/page_6.png`

> [!TIP]
> 이미지 파일 이름은 `data.json`에 정의된 페이지 번호를 따릅니다. "찾아보기" 이미지의 오른쪽 숫자가 해당 페이지 번호입니다.

## 기능
- **스마트 검색**: 건물명만 쳐도 되고, 호실 번호(예: 201)를 치면 자동으로 2층 도면을 찾아줍니다.
- **줌 기능**: 도면을 확대/축소하여 자세히 볼 수 있습니다.
- **자동 완성**: 건물명을 입력할 때 추천 목록이 뜹니다.
